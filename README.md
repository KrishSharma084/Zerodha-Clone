# Zerodha Clone (Full Stack Project)

This project contains three parts:

## 📁 Frontend
- React + Vite
- User trading interface

## 📁 Dashboard
- Admin panel
- Charts and analytics

## 📁 Backend
- Node.js + Express
- MongoDB API server

---

## 🚀 How to run

### Dashboard
cd dashboard
npm install
npm run dev

---

### Frontend
cd frontend
npm install
npm run dev

---

### Backend
cd backend
npm install 
npx nodemon
